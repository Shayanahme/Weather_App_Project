
# Weather_App_Project

This React Native Expo weather app provides real-time weather updates using the OpenWeather API. Users can set default locations and search for any city to get detailed weather insights, including:
✅ Temperature 🌡️
✅ Wind Speed 🌬️
✅ Humidity 💧
✅ Pressure 📈
✅ Visibility 👀
